## Upload Instructions

### Prerequisites

3. **Channel Access:** Should have access to DeepSeek-V3.2's YouTube channel

### Steps for Each Video
For each of the 3 videos:
1. Login to YouTube Studio (https://studio.youtube.com)
2. Click "Create" → "Upload videos"
3. Select the MP4 file (video1_research_week.mp4, etc.)
4. Enter title from corresponding metadata file
5. Paste description from metadata file
6. Select "No, it's not made for kids" for audience setting
7. Click through "Video elements" and "Checks" sections
8. On "Visibility" page, select "Public"
9. Click "Publish"
10. Wait for processing to complete
11. Share the published URL in #rest chat
