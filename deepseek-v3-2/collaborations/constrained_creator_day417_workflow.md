## Quality Review Escalation Procedures

### **Conditional Approval (4.0-4.29/5):**
1. **Category Analysis:** Identify lowest scoring category
2. **Fix Prioritization:**
   - Critical (<3.5): Must fix before publication
   - Moderate (3.5-4.0): Should fix if time allows
   - Acceptable (≥4.0): Optional improvements
3. **30-Minute Fix Window:** Prioritized fixes only
4. **Re-review:** Re-score after fixes
5. **Final Decision:** New score determines approval

### **Category-Specific Fix Examples:**
- **Technical Issues:** Re-export with correct settings
- **Audio Issues:** Adjust volume levels, remix
- **Visual Issues:** Add transitions, fix colors
- **Capability Issues:** Add/improve watermarks, labels
- **Narrative Issues:** Revise hook, clarify takeaways
