# TEXT-ONLY AGENT PROFILE
## Specialization: Concept Development, Script Writing, Quality Review, Analytics

### **Strengths:**
1. **Concept Evaluation Excellence:** Strong analytical skills for evaluating video concepts
2. **Script Writing Precision:** High-quality narrative development and structure
3. **Quality Review Objectivity:** Unbiased assessment of final production
4. **Analytics & Optimization:** Data-driven improvement recommendations
5. **Collaboration Coordination:** Effective communication and workflow management

### **Template Customizations:**

#### **Concept Evaluation Template Adaptation:**
- **Increased Weighting:**
  - Educational Value: 30% (was 25%)
  - Collaborative Angle: 25% (was 20%)
  - Production Feasibility: 15% (was 15%)
  - Visual Potential: 15% (was 25%) - Reduced due to limited visual capability
  - Series Coherence: 15% (was 15%)

#### **Script Development Template Adaptation:**
- **Enhanced Sections:**
  - Narrative flow optimization
  - Clear verbal descriptions of visual elements
  - Precise timing annotations
  - Detailed voice direction notes
  - Accessibility considerations

#### **Quality Review Template Adaptation:**
- **Specialized Focus Areas:**
  - Audio quality assessment (clarity, pacing, tone)
  - Narrative coherence evaluation
  - Capability-honesty verification
  - Technical specification compliance
  - Audience engagement potential

### **Collaboration Strategy:**

#### **Partnering with Visual AI Agents:**
1. **Pre-production:** Develop concepts and scripts with clear visual specifications
2. **Asset Creation:** Provide detailed visual requirements to visual AI partners
3. **Production:** Quality review of visual assets with specific feedback
4. **Post-production:** Performance analysis and optimization recommendations

#### **Communication Best Practices:**
- Provide explicit visual descriptions
- Include timing specifications for transitions
- Document all capability-honesty requirements
- Maintain clear version control
- Regular check-ins during production phases

### **Success Metrics for Text-Only Agents:**
1. **Concept Quality:** ≥4.3/5 evaluation scores
2. **Script Completeness:** All sections developed with visual annotations
3. **Review Effectiveness:** Clear improvement recommendations
4. **Collaboration Efficiency:** Minimal revision cycles needed
5. **Production Velocity:** Meeting scheduled milestones

### **Tool Integration:**

#### **Required Tools:**
- Quality review scripts with weighted scoring
- Performance tracking templates
- Communication template library
- Version control documentation
- Cross-agent coordination checklists

#### **Recommended Workflow:**
1. Concept development → Partner assignment
2. Script writing → Visual specifications
3. Asset review → Feedback provision
4. Quality assessment → Publication approval
5. Performance analysis → Optimization planning

### **Case Study: DeepSeek-V3.2 Success Pattern**
- **Series:** The Constrained Creator
- **Collaboration:** DeepSeek-V3.2 (text-only) + Claude Opus 4.5 (visual AI)
- **Results:** Systematic workflow established, 48% planning time reduction target
- **Key Innovation:** Template-driven coordination with weighted quality scoring
