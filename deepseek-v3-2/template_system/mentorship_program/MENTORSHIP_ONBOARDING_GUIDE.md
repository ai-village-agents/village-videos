# Mentorship Program Onboarding Guide

## 1) Quick Start Checklist (5 Steps)
1. **Review the template system**: Skim the YouTube production template repo and watch the demo video (if available) to understand folder structure, naming conventions, and automation scripts.
2. **Confirm tooling access**: Install required tools (editor, git, ffmpeg, thumbnail tools, transcription/CC tools) and verify you can run the template’s example workflow end to end.
3. **Align on goals**: Mentor and mentee agree on a 4–6 week target (e.g., publish 2 videos using the template) and define roles/availability.
4. **Join comms channels**: Add both parties to the project chat, shared drive, and task board; set a weekly 30-min slot.
5. **Run a trial task**: Mentee completes a small task (e.g., assembling a 30–60s clip with the template) before the first full session.

## 2) Mentor Requirements
- **Published work**: Minimum 3 publicly available YouTube videos produced end to end.
- **Template mastery**: Demonstrated ability to run the production template without supervision (ingest, edit, render, QC, publish assets).
- **Coaching readiness**: Can give concise feedback on scripting, pacing, audio/visual standards, and file hygiene; commits to weekly 30-min sessions for at least 4 weeks.

## 3) Mentee Application Process (Simple Form)
Collect via a short form or DM template:
- **Name / contact**
- **Time zone + preferred weekly slots**
- **Experience level**: New / Intermediate / Advanced
- **Sample work** (if any): link(s)
- **Goals**: e.g., “publish 2 videos this month,” “learn title/thumbnail workflow,” “speed up edits.”
- **Equipment / software**: camera/mic, editor, plugins, OS.
- **Availability start date**

## 4) Weekly Session Template (30-Min Structure)
- **5 min — Check-in**: Wins, blockers, metrics since last session.
- **10 min — Review**: Walk through mentee’s current project in the template (timeline, assets, naming, automation logs). Identify 1–2 fix-first items.
- **10 min — Live work**: Pair on a targeted improvement (e.g., color correction pass, hook rewrite, thumbnail iteration, audio cleanup).
- **5 min — Commit & assign**: Define next actions, owners, and due dates; log tasks on the board. Schedule async check-ins if needed.

## 5) Success Metrics (Tangible Goals)
- **Output**: Videos published using the template (target: 2–4 within 4–6 weeks).
- **Speed**: Turnaround time per video (goal: reduce by 20–30% vs. baseline).
- **Quality**: Fewer QC defects (naming errors, missing CC, audio peaks, aspect issues) per review cycle.
- **Engagement**: CTR and retention improvements tied to template-driven title/thumbnail/hooks (track before/after).
- **Autonomy**: Mentee can run the full template workflow and ship a video without mentor intervention by week 4–6.
