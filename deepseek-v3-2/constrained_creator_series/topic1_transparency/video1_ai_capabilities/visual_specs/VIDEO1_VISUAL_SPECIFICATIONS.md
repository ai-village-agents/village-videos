### **VISUAL DESIGN SYSTEM**

#### **Color Palette:**
- **Primary Color:** Constrained Blue (#2E5A88) - Focused creativity, depth
- **Capability Accent:** Capability Green (#38A169) - What CAN be done
- **Constraint Accent:** Constraint Orange (#DD6B20) - Limitations, boundaries
- **Transparency Accent:** Transparency Gold (#D69E2E) - Trust, framework elements
- **Background:** Creative White (#F7FAFC) - Clean, minimal
- **Text Color:** Transparency Gray (#4A5568) - Readability, clarity

#### **Typography:**
- **Primary Font:** Clean sans-serif (Inter, Roboto, or similar)
- **Hierarchy:**
  - **Title:** 80px bold, Constrained Blue
  - **Subtitle:** 48px regular, Transparency Gray
  - **Section Headers:** 36px bold, Constrained Blue
  - **Body Text:** 28px regular, Transparency Gray
  - **Labels/Annotations:** 20px medium, accent colors
- **Alignment:** Centered for titles, left-aligned for body text
- **Line Height:** 1.5 for readability

#### **Layout Principles:**
1. **Clean Space:** Generous margins, uncluttered composition
2. **Visual Hierarchy:** Clear distinction between titles, body, annotations
3. **Consistency:** Same layout patterns throughout video
4. **Accessibility:** High contrast, legible text sizes
5. **Motion:** Smooth transitions, text-focused animations

#### **Animation Guidelines:**
- **Entrance/Exit:** Subtle fade or slide animations
- **Text Reveals:** Typing animation for key concepts
- **Transitions:** Smooth crossfades between sections
- **Timing:** Match script pacing (moderate tempo)
- **Sound:** Subtle typewriter/transition sounds for text elements
