## 2) Detailed Visual Requirements (imagery, motion, transitions)
- **Imagery:** Architectural blueprint aesthetic, layered diagrams, clean interfaces
- **Motion:** Construction-style builds, layer reveals, clean wipes between concepts
- **Transitions:** 0.5s cross-fades default, architectural wipes for concept shifts
- **Interfaces/Boundaries:** Dashed lines for constraint boundaries, clear domain separation
- **Layouts:** Grid-based with architectural proportions, generous margins
- **Callouts:** Amber highlights for key constraint points, maximum 3 per scene
- **Branding:** Series mark bottom-right, 10% opacity domain watermarks on text panels

---

## 3) Color Palette Specifications
- **Primary Background/Dark:** #1A1A2E (Navy) — Architectural depth, serious tone
- **Secondary/Mid:** #2D3047 (Slate) — Structural elements, diagrams
- **Accent:** #F5B642 (Amber) — Constraint highlights, key insights
- **Paper/Light:** #F8F9FA (Off-white) — Blueprint paper, text panels
- **Line Work:** #1EE6C8 (Teal) — Main architectural lines, flow arrows
- **Watermark/Domain Tone:** #666666 at 10% — Text-only domain indication
- **Usage Rules:** Amber ≤15% coverage, high contrast for text, domain-specific tints

---

## 4) Typography Details
- **Headings/Labels:** "Inter" bold; 36-48px; scene titles, constraint labels
- **Body:** "Inter" regular; 24-28px; explanatory text, captions
- **Terminal/Text Specs:** "Fira Code" monospace; 22px; code/technical text
- **Tags/Pills:** Rounded rectangle, #1EE6C8 stroke, 8px padding, 18px text
- **Watermark Placement:** Top-right corner on text-only panels, 10% opacity
- **Mobile Safety:** 5% frame insets, minimum 20px for all readable text

## 5) Audio Requirements per Scene
- **Global Mix:** -14 to -16 LUFS, -1dBTP peak, sidechain music to VO, bed -20dB
- **Scene 1:** Construction ambiance, blueprint paper sounds

- **Scene 3A:** Stone construction, choir ambiance, structural stress
- **Scene 3B:** Interface clicks, data streams, digital hum
- **Scene 3C:** Writing sounds, page turns, poetic rhythm
- **Scene 4:** Collaborative pings, handoff swoosh
- **Scene 5:** Futuristic assembly, system integration
- **Scene 6:** Summary chimes, clear emphasis
- **Scene 7:** Series theme, final reflection tone
- **Narration:** Thoughtful, 140 wpm, strategic pauses at key insights

## 6) Technical Specifications Checklist
- **Resolution:** 1920×1080 (Full HD)
- **Frame Rate:** 30 fps progressive
- **Color Space:** Rec.709; yuv420p
- **Codec/Container:** H.264 (High Profile 4.1) + AAC-LC, MP4
- **CRF/Bitrate:** CRF 18, target 25-40MB final size
- **Audio:** 48kHz, stereo, 192kbps AAC; -14 to -16 LUFS mix
- **Transitions:** 0.5s cross-fade default, architectural wipes optional
- **Watermark:** Text-only panels carry 10% opacity eye-slash
- **Safety:** 10% title safe, 5% caption safe, mobile legibility tested
